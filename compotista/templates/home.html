Compotista

Hello, Compotista is a simple webapp that convert the Members of the European Parliament data taken from parltrack in the "representatives" format.

If you aren't interested in those data and/or aren't using a lib/application that is implementing the "representatives" format, this website won't have a lot of interest for you.

Data:

  • Latest json dump of representatives dump
  • Checksum of the data

Compotista is free software under the aGPLv3+. Source code

Garlandus Compotista was a Belgium logician.
